# Bayesian Network Online Prediction

这是一个用于展示和交互式使用贝叶斯网络模型的网页。用户可以点击图谱中的节点设置证据，网页会自动更新所有节点的后验概率，并重点展示 PPCM 的预测概率。

## 在线访问

GitHub Pages 链接：

```text
https://cherryhhhh.github.io/wangye/
```

如果链接暂时打不开，可能是 GitHub Pages 还在部署中，等待几分钟后刷新即可。

## 网页功能

- 展示完整贝叶斯网络图谱。
- 点击任意节点可设置该节点的证据值。
- 设置证据后，所有节点概率会自动更新。
- 每个节点都会显示所有状态的概率。
- PPCM 节点固定显示在图谱最下方，方便查看最终预测结果。
- 右侧面板会列出当前证据和所有节点的后验概率。

## 使用方法

1. 打开网页链接。
2. 点击图谱中的任意节点。
3. 在弹出的窗口中选择该节点的状态作为证据。
4. 网页会自动重新计算并显示所有概率。
5. 如需清除证据，可点击右侧面板中的清除按钮。
